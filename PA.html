Conceitos Fundamentais

O que é um Algoritmo?

Um algoritmo é uma sequência finita de passos bem definidos que resolvem um problema ou realizam uma tarefa específica. É como uma receita de bolo: uma série de instruções que, quando seguidas corretamente, produzem um resultado esperado.

💡 Exemplo prático: Um algoritmo para fazer café:
  1. Ferva a água
  2. Coloque o pó de café no filtro
  3. Despeje a água quente sobre o pó
  4. Aguarde o café coar
  5. Sirva em uma xícara

Algoritmos Clássicos

1. Algoritmo de Busca Binária

Um algoritmo eficiente para encontrar um elemento em uma lista ordenada. Ele funciona dividindo repetidamente pela metade a parte da lista que pode conter o elemento.

function buscaBinaria(array, elemento) {
    let inicio = 0;
    let fim = array.length - 1;
    
    while (inicio <= fim) {
        let meio = Math.floor((inicio + fim) / 2);
        
        if (array[meio] === elemento) {
            return meio; // Elemento encontrado
        }
        
        if (array[meio] < elemento) {
            inicio = meio + 1; // Busca na metade direita
        } else {
            fim = meio - 1; // Busca na metade esquerda
        }
    }
    
    return -1; // Elemento não encontrado
}

2. Algoritmo de Ordenação Bubble Sort

Um algoritmo simples de ordenação que percorre a lista múltiplas vezes, comparando elementos adjacentes e trocando-os se estiverem na ordem errada.

function bubbleSort(array) {
    let n = array.length;
    
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            if (array[j] > array[j + 1]) {
                // Troca os elementos
                let temp = array[j];
                array[j] = array[j + 1];
                array[j + 1] = temp;
            }
        }
    }
    
    return array;
}

3. Algoritmo de Fibonacci

Gera a sequência de Fibonacci, onde cada número é a soma dos dois anteriores: 0, 1, 1, 2, 3, 5, 8, 13, ...

function fibonacci(n) {
    if (n <= 1) return n;
    
    let a = 0, b = 1;
    
    for (let i = 2; i <= n; i++) {
        let temp = a + b;
        a = b;
        b = temp;
    }
    
    return b;
}

// Versão recursiva
function fibonacciRecursivo(n) {
    if (n <= 1) return n;
    return fibonacciRecursivo(n - 1) + fibonacciRecursivo(n - 2);
}
⚠️ Importante: A versão recursiva do Fibonacci é ineficiente para valores grandes devido à repetição de cálculos. A versão iterativa é muito mais rápida!

Estruturas de Dados

Listas/Arrays

Coleções ordenadas de elementos que podem ser acessados por índice. Ideais para armazenar sequências de dados.

Operações: inserir, remover, acessar, buscar

Pilhas (Stack)

Estrutura LIFO (Last In, First Out) - o último elemento inserido é o primeiro a sair. Útil para desfazer operações, navegação em navegadores.

Filas (Queue)

Estrutura FIFO (First In, First Out) - o primeiro elemento inserido é o primeiro a sair. Usado em sistemas de impressão, processamento de tarefas.

Árvores

Estrutura hierárquica com nós e conexões. Árvores binárias são usadas em busca eficiente, expressões matemáticas.

Grafos

Conjunto de vértices conectados por arestas. Modelam redes sociais, mapas de rotas, sistemas de recomendação.

Tabelas Hash

Estrutura que mapeia chaves a valores para acesso rápido. Usado em bancos de dados, caches, dicionários.
